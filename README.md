# WinScreenLayout

WinScreenLayout is a lightweight Windows tool to save and restore window and monitor layouts.

It is designed for reproducible setups across reboots, docking stations, and multi-monitor environments.

## Features
- Save current window & monitor layout
- Restore layouts reliably
- CLI-first, scriptable
- No background services
- No cloud, no telemetry

## Use cases
- Laptop + docking station
- Office / Home / Gaming profiles
- Admin scripts & automation
- Reproducible workstation setups

## Components
- **Core**: layout detection & apply logic
- **CLI**: scripting and automation interface

## Status
- Actively developed
- Early but usable
- Interfaces may evolve

## Philosophy
- Local-first
- Transparent
- Minimal dependencies
- Open by default
